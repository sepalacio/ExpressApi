# ExpressApi
API created with express framework, nodeJS, mongoDB and Mongoose ORM

##Install dependencies
 `$ npm install`

##Run proyect

######Initialize MongoDB
`$ mongod`

######Run Node server
`$ node app.js`

##Requires
1. Node v4.2.1 or higher due to arrow functions (Ecmascript 2015).
2. MongoDB as database server.
